Invitations

Create Individual Page Invitation

Send an invitation to create an individual supporter page to an email address.

POST /individual-pages/invitations

Visibility

Restricted. Requires an authenticated user.

Payload

email
Required string. The email address you would like the invitation sent to. We do not tie this email address to the user’s Giving Passport in any way, it is only used to deliver the invitation.
phone
Optional string. The phone number of the person you want to invite to create a supporter page.
full_name
Optional string. The full name of the person you want to invite to create a supporter page.
charity_id
Optional string. The id of the charity that you want set for this supporter page. If no charity is supplied the user will be prompted to select a charity. Defaults to none.
target
Optional decimal. The amount of money this page is aiming to raise. Defaults to a predetermined value.
{
  "email": "support@everydayhero.com",
  "phone": "1300798768",
  "full_name": "Everyday Hero Support",
  "charity_id": "au-1",
  "target": 705.5
}

Response

{
  "create_individual_page_invitation": {
    "id": 1,
    "email": "support@everydayhero.com",
    "accepted_at": "2013-04-01T00:00:00Z",
    "cancelled_at": null
  }
}

Restricted. Requires an authenticated user.

full_name
Optional string. The full name of the person you want to invite to create a supporter page.
charity_id
Optional string. The id of the charity that you want set for this supporter page. If no charity is supplied the user will be prompted to select a charity. Defaults to none.
target
Optional decimal. The amount of money this page is aiming to raise. Defaults to a predetermined value.
{

  "target": 705.5
